# Glint Native Slides

**_A liquid-smooth local viewer for PowerPoint (.pptx) files_**

Glint is a native slide renderer for Obsidian. Built with a focus on aesthetics and frictionless performance, it allows you to view your presentation assets directly in your vault without clunky PDF conversions or external software.
