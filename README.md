# Glint Native Slides

A liquid-smooth local viewer for PowerPoint (.pptx) files.

![License](https://img.shields.io/github/license/bohem-dev/glint-native-slides)
![Version](https://img.shields.io/github/v/release/bohem-dev/glint-native-slides)

## Installation

1. Install.
2. Enable "Glint Native Slides" in settings.
3. Click any `.pptx` file in your file explorer to view it.

## Development

```bash
npm install
npm run dev
